# Copy Of Trello

Proyecto de aprendizaje de una aplicación web recreando desde 0 trello, usando buenas practicas de desarrollo e ingeniería de software.

## Tecnologías
- **Backend:** Django (Python)
- **Froend**: Vite (Vanilla)
- **DB**: Maria DB
